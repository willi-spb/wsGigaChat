Коннектор к gigachat - для Delphi XE8 Berlin 32bit app (используется Indy - т.к. только после версии Rio стандартный ThttpClient поддерживает протокол https.
Реализован минимум нужных методов (пока с 1 сообщением и 1 картинкой).

Особенности:
1. В проекте есть каталог общих модулей, в котором модули логирования, преобразований с Json и прочее
2. Проект содержит код free Json-библиотеки XSuperObject - исходник можно найти на github.com
3. Проект содержит диалог-форму с TcxGrid и TmemoryData для преобразования списка файлов гигачата в таблицу с возможностью удаления файлов,
для этого диалога требуется установленная библиотека devExptess for Delphi - если она не установлена - исключите форму и ссылку из проекта
